# 07_Resume_Portfolio_Site

Claude Code 강의 Chapter 7 결과물입니다. PRD와 자연어 프롬프트를 바탕으로 만든 개인 이력서/포트폴리오 웹사이트입니다.

## 프로젝트 개요

- 목적: 본인 소개, 기술 스택, 프로젝트, 경력, 강의 이력, 수상 이력을 한 페이지에 정리하는 포트폴리오 사이트 제작
- 학습 포인트: PRD 작성, Claude Code 계획 수립, 구현, 디자인 시스템 커스터마이징, GitHub/Vercel 배포
- 원본 레포: https://github.com/swan-crypto/landing

## 주요 기능

- 히어로 섹션과 상단 내비게이션
- 기술 스택, 프로젝트, 경력, 강의, 수상, 연락처 섹션
- 반응형 포트폴리오 레이아웃
- `docs/PRD.md`, `DESIGN.md`, `CLAUDE.md` 기반 작업 지침 포함

## 기술 스택

- Next.js
- React
- TypeScript
- Tailwind CSS

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 열어 확인합니다.

## 주요 명령어

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## 강의 활용 포인트

이 프로젝트는 Claude Code로 PRD를 작성하고, 계획 수립 후 구현과 디자인 커스터마이징까지 이어지는 기본 흐름을 보여주는 Chapter 7 참고 결과물입니다.
